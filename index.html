功能介紹

一個協定，打通所有企業數據

基於開放的 MCP 協定，安全地將企業資料源接入 AI 助手

Google Analytics 整合

直接透過 AI 助手查詢 GA4 報表數據，包括即時報表、標準報表、帳戶摘要與自訂維度，不需要再開 GA 介面。

OAuth 安全授權

每位員工各自授權自己的 Google 帳號，憑證加密儲存，不共用密碼，符合企業資安要求。

自然語言查詢

不用學 GA 介面操作，直接問「上週的轉換率多少？」AI 助手會自動呼叫正確的 API 並整理結果回覆你。

權限動態管理

沒有授權 GA 的員工，AI 助手自動隱藏 GA 相關工具。管理者可隨時開關員工的 MCP 存取權限。

即插即用

相容 Claude Desktop、Cursor、Windsurf 等所有支援 MCP 的 AI 工具，複製設定 JSON 即可連線使用。

可擴展架構

目前支援 Google Analytics，未來可擴展 GitHub、AWS、Slack 等更多整合，統一入口管理所有數據工具。

使用方式

三步驟開始使用

從註冊到查詢，只需要幾分鐘

1

啟用 MCP 並取得金鑰

登入 EIP 系統，在 MCP 設定頁面啟用功能，系統會自動產生你的專屬認證金鑰。

2

授權 Google Analytics

在整合頁面點擊「連結 Google Analytics」，透過 Google OAuth 安全授權你的 GA 帳號。

3

開始對話查詢

將 MCP 設定 JSON 貼到 Claude Desktop，就能直接用自然語言向 AI 詢問你的 GA 數據。
