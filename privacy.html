隱私權與政策

最後更新日期：2026 年 3 月

隱私權保護政策的適用範圍

非常歡迎您使用「光速火箭股份有限公司」（以下簡稱本公司）提供之網站及相關服務。本隱私權保護政策說明本公司如何處理在您使用服務時收集到的個人識別資料。本政策不適用於本公司網站以外的相關連結網站，也不適用於非本公司所委託或參與管理的人員。

個人資料的蒐集、處理及利用方式

  • 當您造訪本公司網站或使用本公司所提供之功能服務時，我們將視該服務功能性質，請您提供必要的個人資料，並在該特定目的範圍內處理及利用您的個人資料；非經您書面同意，本公司不會將個人資料用於其他用途。
  • 本公司在您使用服務信箱、問卷調查等互動性功能時，會保留您所提供的姓名、電子郵件地址、聯絡方式及使用時間等。
  • 於一般瀏覽時，伺服器會自行記錄相關行徑，包括您使用連線設備的 IP 位址、使用時間、使用的瀏覽器、瀏覽及點選資料記錄等，做為我們增進網站服務的參考依據，此記錄為內部應用，決不對外公佈。
  • 為提供精確的服務，我們會將收集的問卷調查內容進行統計與分析，分析結果之統計數據或說明文字呈現，除供內部研究外，我們會視需要公佈統計數據及說明文字，但不涉及特定個人之資料。
  • 您可以隨時向我們提出請求，以更正或刪除本公司所蒐集您錯誤或不完整的個人資料。

Google API 服務資料使用揭露

本公司之企業內部系統（EIP）整合了 Google API 服務，以提供員工 Google Analytics 數據查詢功能。以下說明本公司如何存取、使用、儲存及分享透過 Google API 取得之使用者資料：

存取的資料範圍

本公司透過 Google OAuth 2.0 授權機制，僅請求以下權限範圍（Scopes）：

  • analytics.readonly：以唯讀方式存取使用者已授權之 Google Analytics 4 資產的報表數據，包括帳戶摘要、資產詳情、標準報表及即時報表。
  • userinfo.email：取得使用者的電子郵件地址，僅作為帳號識別用途。

資料使用方式

透過 Google API 取得之資料，僅用於讓經授權之員工透過本公司內部系統的 MCP（Model Context Protocol）介面，查詢其有權存取之 Google Analytics 報表數據。使用者的電子郵件地址僅作為授權帳號的識別標記。

本公司不會將透過 Google API 取得之資料用於廣告投放、市場研究、資料轉售，或任何與上述目的無關之用途。本公司不會將透過 Google API 取得之資料用於訓練人工智慧或機器學習模型。

資料儲存方式

  • OAuth 授權憑證（access token 及 refresh token）以加密方式儲存於本公司內部伺服器的資料庫中，僅供系統存取 Google API 時使用。
  • Google Analytics 報表數據不會被本公司永久儲存；每次查詢皆即時向 Google API 請求，查詢結果僅於該次操作期間暫存。

資料分享

本公司不會與任何第三方分享透過 Google API 取得之使用者資料，除非經使用者明確書面同意，或法律明文規定及司法機關依法要求。

使用者權利

  • 使用者可隨時透過本公司內部系統（EIP）撤銷 Google 帳號授權，撤銷後系統將刪除該帳號之所有 OAuth 憑證。
  • 使用者亦可透過 Google 帳號安全設定頁面直接移除本應用程式的存取權限。

合規聲明

本公司對 Google API 服務資料之使用，遵循 Google API 服務使用者資料政策，包括有限使用（Limited Use）之相關規範。

資料之保護

  • 本公司主機均設有防火牆、防毒系統等相關的各項資訊安全設備及必要的安全防護措施，加以保護網站及您的個人資料採用嚴格的保護措施，只由經過授權的人員才能接觸您的個人資料，相關處理人員皆簽有保密合約，如有違反保密義務者，將會受到相關的法律處分。
  • 如因業務需要有必要委託其他單位提供服務時，本公司亦會嚴格要求其遵守保密義務，並且採取必要檢查程序以確定其將確實遵守。

網站對外的相關連結

本公司網站的網頁提供其他網站的網路連結，您也可經由本公司網站所提供的連結，點選進入其他網站。但該連結網站不適用本公司的隱私權保護政策，您必須參考該連結網站中的隱私權保護政策。

與第三人共用個人資料之政策

本公司絕不會提供、交換、出租或出售任何您的個人資料給其他個人、團體、私人企業或公務機關，但有法律依據或合約義務者，不在此限。前項但書之情形包括不限於：

  • 經由您書面同意。
  • 法律明文規定。
  • 為免除您生命、身體、自由或財產上之危險。
  • 與公務機關或學術研究機構合作，基於公共利益為統計或學術研究而有必要，且資料經過提供者處理或蒐集者依其揭露方式無從識別特定之當事人。
  • 當您在網站的行為，違反服務條款或可能損害或妨礙網站與其他使用者權益或導致任何人遭受損害時，經網站管理單位研析揭露您的個人資料是為了辨識、聯絡或採取法律行動所必要者。
  • 有利於您的權益。
  • 本公司委託廠商協助蒐集、處理或利用您的個人資料時，將對委外廠商或個人善盡監督管理之責。

Cookie 之使用

為了提供您最佳的服務，本公司會在您的電腦中放置並取用我們的 Cookie，若您不願接受 Cookie 的寫入，您可在您使用的瀏覽器功能項中設定隱私權等級為高，即可拒絕 Cookie 的寫入，但可能會導致網站某些功能無法正常執行。

隱私權保護政策之修正

本公司隱私權保護政策將因應需求隨時進行修正，修正後的條款將刊登於網站上。
